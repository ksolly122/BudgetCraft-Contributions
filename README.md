# BudgetCraft – My Contributions

## Overview

BudgetCraft is a team-based personal finance application built using Ionic React, TypeScript, and Firebase. The app allows users to track income, expenses, and savings goals through a clean and interactive interface.

I contributed to this project as part of a development team, focusing on testing, feature development, and UI improvements.

---

## My Contributions

### Testing (Cypress + Unit Testing)

* Built and improved Cypress end-to-end tests to validate real user flows
* Wrote unit tests to verify application logic and UI behavior
* Fixed and debugged test cases to improve reliability

### Feature Development

* Implemented transaction creation functionality
* Built search and filtering system for transactions
* Developed custom subcategory add/delete logic

### UI / Frontend

* Designed and styled the home page layout
* Created and improved overall page layout structure
* Fixed UI issues including date picker behavior

### Backend / Data Work

* Helped connect the application to the database
* Worked with data handling and state updates
